# Global Green Energy Report 2026

## Project Overview
This project is a semantic HTML technical report focused on renewable energy trends and performance in 2026. The webpage demonstrates proper HTML structure, accessibility practices, and SEO-friendly content organization.

The report includes:
- Semantic HTML layout
- Internal navigation links
- A structured table of contents
- Renewable energy data tables
- Key findings and takeaways

---

## Features

### Semantic Structure
The project uses semantic HTML elements for accessibility and organization:

- `<header>`
- `<nav>`
- `<main>`
- `<article>`
- `<section>`
- `<footer>`

### Accessible Navigation
- Internal anchor links
- Organized table of contents
- Clear reading hierarchy

### Structured Content
- Proper heading levels (`h1`, `h2`, `h3`)
- Data table with captions and table sections
- Lists for summaries and takeaways
